# F&F Corporation Q4 2025 Dashboard

F&F Corporation의 2025년 4분기 재무 대시보드입니다.

## 설치 및 실행

### 1. 의존성 설치
```bash
npm install
```

### 2. 개발 서버 실행
```bash
npm run dev
```

브라우저에서 `http://localhost:5173`으로 접속하시면 대시보드를 확인하실 수 있습니다.

### 3. 빌드
```bash
npm run build
```

## 주요 기능

- **전체요약 탭**: 손익 요약, 재무상태 요약, 경쟁사 비교, AI 분석
- **손익계산서 탭**: 상세 손익계산서 및 법인별 분석

## 기술 스택

- React 18
- Recharts (차트 라이브러리)
- Tailwind CSS (스타일링)
- Vite (빌드 도구)

## 참고사항

현재 모든 데이터는 0으로 설정되어 있습니다. 실제 데이터를 입력하시면 대시보드가 업데이트됩니다.
